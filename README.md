# 🔐 PASSGEN – Password Generator & Manager

A modern and secure **Password Generator & Vault Web App** built using React and Node.js.
Generate strong passwords instantly and store them securely with an intuitive dashboard.

---

## 🚀 Features

* 🔑 **Random Password Generator**

  * Custom length (6–20)
  * Include numbers & symbols
  * One-click refresh 🔄

* 📋 **Copy to Clipboard**

  * Instantly copy generated passwords

* 💾 **Password Vault**

  * Save and manage your passwords
  * Organized with groups

* 🗑 **Recycle Bin**

  * Recover deleted passwords
  * Permanent delete option

* 📊 **Dashboard**

  * Total passwords count
  * Group-wise distribution
  * Recently added passwords

* 👤 **Authentication**

  * Secure login/signup
  * Account management
  * Delete account option

* 🎨 **Modern UI**

  * Glassmorphism design
  * Icon-based actions
  * Responsive layout

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/your-username/passgen.git
cd passgen
```

### 2️⃣ Install dependencies

#### Frontend

```
cd client
npm install
npm run dev
```

#### Backend

```
cd server
npm install
npm start
```

---

## 🔑 Environment Variables

Create a `.env` file in the server folder:

```
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
PORT=5000
```
